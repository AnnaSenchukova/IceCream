## Примеры промптов
#### CSS АРХИТЕКТУРА И НЕЙМИНГ (фрагмент)
Используй БЭМ-именование классов в международной нотации:
- lowercase Latin
- слова через дефис
- элемент через __
- модификатор через --

#### Архитектурный рефакторинг 1200px
«на 1200 убрать дубль правил — все под один брекпоинт; убрать отступы/паддинги вокруг билдера; билдер как на таблете 768 и ~35–40% экрана; каталог — увеличить цены; по 3 вкуса в ряд».

### Чистка и code-style
«убрать все комментарии кроме заголовков блоков; проверить на дубли в разных медиазапросах; убрать неработающие/избыточные стили; сгруппировать builder-правила в один блок внутри каждого медиазапроса; вынести max-width в самый конец».
